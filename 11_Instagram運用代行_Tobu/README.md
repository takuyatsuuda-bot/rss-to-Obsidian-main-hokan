# Instagram運用代行 - Tobu Top Tours

## 概要
東武トップツアーズ様のInstagram公式アカウントの運用代行業務に関するフォルダです。
日々のフィード・リール投稿の管理、および年4回のフォトコンテストの運営管理を行います。

- **公式アカウント**: [tobutoptours_official](https://www.instagram.com/tobutoptours_official/)
- **主な目的**: 
    - 継続的な投稿によるフォロワー増加
    - フォトコンテストを通じたUGC（ユーザー生成コンテンツ）の創出とコミュニティ活性化

## フォルダ構成
- **Operation_Manual.md**: 日々の運用ルール、投稿フォーマット
- **Content_Planning.md**: 投稿ネタの企画メモ
- **Photo_Contest/**: フォトコンテスト関連フォルダ
    - **Contest_Workflow.md**: コンテストの流れ・手順
    - **Past_Winners_Database.md**: 過去の入賞者管理リスト
