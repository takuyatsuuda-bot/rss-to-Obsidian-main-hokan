# 動画素材フォルダの使い方

このフォルダは、Instagram運用代行で使用する動画素材（クイズ動画のついでに撮影した旅行動画など）を蓄積するためのものです。

## フォルダ構成

フォルダは「地域」＞「スポット名」の階層で作成してください。

例：
```
03_動画素材/
  ├── 動画素材管理表.md  <-- これが重要！
  ├── README.md
  ├── 関西地方/
  │   ├── 大阪城/
  │   │   ├── DJI_001.mp4
  │   │   └── IMG_9923.mov
  │   └── 通天閣/
  └── 関東地方/
      └── 東京タワー/
```

## 運用ルール

動画素材（実データ）は**外付けディスク**に保存される予定とのことですので、このフォルダには管理表だけを置く形になります。

### 管理表の更新方法（2パターン）

1.  **Chatで報告して更新する（おすすめ）**
    *   チャットで「〇〇地方の△△の動画を撮ってきたよ。ファイル名は `DSC_001.mov` で、外付けHDDの `2026_Travel` フォルダに入れた」のように報告してください。
    *   私がそれを聞き取って、自動で `動画素材管理表.md` を更新します。

2.  **手動で記入する**
    *   `動画素材管理表.md` を直接開き、行を追加してください。

## フォルダ構成（管理表のみ）

```
03_動画素材/
  ├── 動画素材管理表.md  <-- これが重要！
  └── README.md
```
※ 実データは外付けディスクにあるため、ここには地域ごとのフォルダは必須ではありませんが、もしObsidian内でも整理したい場合は作っても構いません。

## AIへの聞き方例

チャットで以下のように質問すると、このフォルダと管理表を探して回答します。

*   「関西地方の動画素材、何かある？」
*   「東京タワーの動画ってストックあるかな？」
*   「海が写ってる動画素材を探して」
