# 🎮 FIFA26 (EA SPORTS FC) ゲーム配信開始プロジェクト

## 1. プロジェクトの目的（Why）
- **罪悪感のない趣味時間:** 「ただの浪費」ではなく「コンテンツ制作（生産活動）」とすることで、堂々とゲームを楽しむ。
- **スキル資産化:** 配信環境の構築、動画編集、AI活用の知見をビジネス（SNS運用・コンサル）に還元する。
- **経費化の検討:** 事業活動の一環として取り組み、機材を経費計上することで節税効果を狙う（※税理士確認）。

## 2. 資金調達計画（Funding）
**目標金額：約130,000円**
- **資金源:** 日興フロッギー（dポイント投資）
- **戦略:** 1. 現在〜8月：普段のポイ活で貯めたdポイントを日興フロッギーで運用し続ける。
    2. 2026年9月：運用益を含めた株式を売却し、現金化する。
    3. 楽天スーパーSALE等のタイミングで機材を購入（さらにポイント獲得）。

## 4. 配信スタイル・コンテンツ戦略
- **基本方針:** 「喋り」ありの配信（プレイスキル補完のため）。
- **差別化案:** - 「素の声」を出したくない場合は、AI音声（ずんだもん等）やボイスチェンジャーを検討。
    - 「AIおぢ」キャラの活用も視野に入れる。
- **収益・メリット:** - YouTube収益（ハードル高）よりも、まずは「Amazonアソシエイト（ガジェット紹介）」や「ポイ活ブログへの流入」を狙う。
