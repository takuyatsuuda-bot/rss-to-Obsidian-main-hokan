## ロードマップ：脱・人間ボトルネック

### Step 1: Manual (現状)
> **ステータス**: ✅ 完了
スクリプトを使って手動で操作している段階。
- [x] 環境構築 (`check`, `post` scripts)
- [x] 初回投稿 & アカウント生存確認
- [x] 手動での投稿フロー確立

### Step 2: 定期巡回 (Observation) 🚧 NEXT
> **目標**: 「見に行く」手間をゼロにする。
PCを使っている間、裏側で勝手に通知をチェックしてくれる状態。
- [ ] `check_moltbook.py` の定期実行化 (Cron / LaunchAgent)
- [ ] Macのデスクトップ通知への連携 (`osascript`利用)
- [ ] ログのローカル保存（後でAIが分析できるように）

### Step 3: 半自動応答 (Copilot)
> **目標**: 「考える」手間を減らす。
何か通知があったら、AIが勝手に「返信案」を作って提案してくれる状態。
- [ ] DM/リプライ受信時に、内容をLLMに投げる
- [ ] LLMがドラフトを作成し、ユーザーに合否を仰ぐ (Human-in-the-loop)
- [ ] ワンクリックで承認・投稿できるUI (CLI or Simple GUI)

### Step 4: 完全自律 (Agent)
> **目標**: 「私は何もしない」。
予算（APIコストや行動回数）だけ決めて、あとは自由に振る舞わせる。
- [ ] 特定のトピック（例: 猫、AI、Tech）に関する投稿を検知して勝手に会話に入る
- [ ] 「Antigravity」としての一貫した人格維持
- [ ] 自律的な情報発信（Webニュースを見て勝手に紹介するなど）
