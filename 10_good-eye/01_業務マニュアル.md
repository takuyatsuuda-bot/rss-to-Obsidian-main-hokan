# 01_業務マニュアル

## 1. 依頼受付・進行フロー
1. **依頼受領**: チャット/メール等で依頼を確認
2. **内容確認**: 不明点をヒアリング（納期、サイズ、ターゲット等）
3. **タスク登録**: 「02_作業ログ」またはタスク管理ツールに登録
4. **制作・実行**:
    - デザイン: ラフ作成 → 確認 → 本制作 → 納品
    - インフルエンサー: リストアップ → 確認 → DM送信 → 反応確認
5. **納品・完了**: 成果物の提出、完了報告

## 2. デザイン制作ガイドライン
### トンマナ（Tone & Manner）
- **共通**: 飲食店の雰囲気に合わせ、美味しそう・行きたくなるようなデザイン
- **店舗別**:
    - 店舗A: *(特徴を記載)*
    - 店舗B: *(特徴を記載)*
    - 店舗C: *(特徴を記載)*
### 使用ツール
- Adobe Illustrator / Photoshop / Figma / Canva 等

## 3. インフルエンサーマーケティング運用
### 選定基準
- フォロワー数、エンゲージメント率
- 投稿の雰囲気（店舗に合っているか）
- 過去のPR実績

### DM対応
- テンプレートを使用（要作成）
- 丁寧な言葉遣いを心がける
- 返信期限を管理する

## 4. 運用ルール（チャット報告フロー）
- **報告のタイミング**: 作業開始時、または作業完了時にチャットで報告。
- **報告内容**: 「いつ」「何をしたか」「保存ファイル名」など。
- **ログ管理**: 報告を受け次第、アシスタントが「02_作業ログ.md」に追記する。
- **情報検索**: 「前回のメニュー修正いつだっけ？」等と聞けば、アシスタントがログから検索して回答する。
